## Routes

```sh
  auth:

    POST /signup
    POST /signin
    
  post:

    GET /posts
    GET /posts/:userId
    GET /post/:id
    POST /post
    PUT /post/:id
    DEL /post/:id

  user:

    GET /user
    GET /users
    DEL /user
```
